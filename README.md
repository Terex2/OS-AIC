# Open Source AI Company (OS-AIC) | شركة الذكاء الاصطناعي مفتوحة المصدر

مشروع طموح يهدف إلى دمج أقوى الأدوات مفتوحة المصدر لبناء نظام ذكاء اصطناعي متكامل، مستقل، وقوي. يعتمد المشروع على فلسفة "السيادة التقنية" حيث يتم تشغيل كل شيء محلياً أو على خوادم خاصة دون الاعتماد على خدمات خارجية مدفوعة.

## 🚀 القدرات الرئيسية (Key Capabilities)

| المجال | الأدوات المستخدمة | الوصف |
| :--- | :--- | :--- |
| **التنسيق (Orchestration)** | LangGraph, CrewAI, AutoGen | إدارة الوكلاء المتعددين والمهام المعقدة |
| **النماذج (Models)** | Ollama, vLLM, llama.cpp | تشغيل نماذج اللغة الضخمة محلياً |
| **الويب (Web)** | Browser-use, Crawl4AI, Playwright | التصفح الآلي واستخراج البيانات |
| **المعرفة (RAG)** | Chroma, Qdrant, Docling | ذاكرة طويلة الأمد وفهم المستندات |
| **الوسائط (Media)** | Whisper, Coqui TTS, ComfyUI | معالجة الصوت، الصور، والفيديو |
| **البرمجة (Coding)** | OpenHands, Tree-sitter | تطوير الأكواد البرمجية آلياً |

## 🏗️ هيكل المشروع (Project Structure)

```bash
.
├── core/          # النواة والمحرك الرئيسي (LangGraph/LangChain)
├── agents/        # تعريف الوكلاء المتخصصين (CrewAI/AutoGen)
├── services/      # الخدمات المستقلة (Ollama, API, RAG)
├── data/          # تخزين البيانات وقواعد البيانات (Postgres/SQLite)
├── ui/            # واجهات المستخدم (Open WebUI, Streamlit)
└── docs/          # التوثيق الفني والمعمارية
```

## 🛠️ البدء (Getting Started)

يتطلب المشروع وجود **Docker** و **Python 3.10+**.

1. قم بتحميل المستودع.
2. قم بتثبيت المتطلبات: `pip install -r requirements.txt`.
3. قم بتشغيل الخدمات الأساسية عبر Docker: `docker-compose up -d`.
