# Open Source AI Company (OS-AIC) v2.0 | شركة الذكاء الاصطناعي مفتوحة المصدر - الإصدار المطور

هذا هو الإصدار "الشيطاني" المطور من مشروع OS-AIC، حيث تم تحويله من مجرد واجهة برمجية بسيطة إلى نظام هجين خارق يمتلك حواساً وقدرات تنفيذية مستقلة. يعتمد النظام الآن على فلسفة **"الاستقلال الذاتي الكامل"**، حيث يمكنه التفكير، البحث في الويب، التعلم من المستندات، وإصلاح أكواده البرمجية ذاتياً.

## 🚀 القدرات الخارقة (Advanced Capabilities)

| القدرة | الأداة | الوصف "الشيطاني" |
| :--- | :--- | :--- |
| **الاستدلال العميق (Reasoning)** | LangGraph + Llama 3 | طبقة تفكير منطقي تحلل المهام قبل تنفيذها |
| **وكيل الويب (Web Infiltrator)** | Playwright + Crawl4AI | القدرة على تصفح الويب واستخراج البيانات وتجاوز العقبات |
| **الذاكرة السيادية (RAG)** | Qdrant + Embedding | ذاكرة طويلة الأمد تفهم سياق مشاريعك وبياناتك الخاصة |
| **منفذ الكود (Auto-Coder)** | Auto-Coder (Custom) | وكيل برمجة داخلي يمكنه كتابة، تشغيل، وإصلاح الكود ذاتياً |
| **التنسيق الهجين (Orchestra)** | Hybrid Orchestrator | محرك ذكي يقرر متى يبحث في الويب ومتى يستخدم الذاكرة |

## 🏗️ هيكل المشروع (Project Structure)

```bash
.
├── core/          # النواة والمحرك الرئيسي (LangGraph/LangChain)
├── agents/        # تعريف الوكلاء المتخصصين (CrewAI/AutoGen)
├── services/      # الخدمات المستقلة (Ollama, API, RAG)
├── data/          # تخزين البيانات وقواعد البيانات (Postgres/SQLite)
├── ui/            # واجهات المستخدم (Open WebUI, Streamlit)
└── docs/          # التوثيق الفني والمعمارية
```

## 🛠️ البدء (Getting Started)

يتطلب المشروع وجود **Docker** و **Python 3.10+**.

1. قم بتحميل المستودع.
2. قم بتثبيت المتطلبات: `pip install -r requirements.txt`.
3. قم بتشغيل الخدمات الأساسية عبر Docker: `docker-compose up -d`.
